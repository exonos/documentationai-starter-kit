## Prerequisites

<Callout kind="info">

Before starting, ensure you have:
- A free apimarket.mx account (sign up takes 1 minute)
- Node.js (for JavaScript examples) or Python 3.8+
- cURL (usually pre-installed)
- Access to the sandbox environment at `https://api.apimarket.mx/sandbox`

</Callout>

## Create Your Account

Follow these steps to register and log in.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Go to [https://dashboard.apimarket.mx](https://dashboard.apimarket.mx) and click **Sign Up**.
  </Step>
  <Step title="Register" icon="user-plus">
    Enter your email, password, and company details. Verify your email.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard and sign in with your credentials.
  </Step>
</Steps>

## Generate Your API Key

API keys power all requests. Generate one in the sandbox mode for testing.

<Steps>
  <Step title="Navigate to API Keys" icon="key">
    In the dashboard, go to **Settings > API Keys**.
  </Step>
  <Step title="Create Sandbox Key" icon="plus">
    Click **Generate New Key**, select **Sandbox Environment**, and copy the key (e.g., `sk_test_your-sandbox-key-here`).
  </Step>
  <Step title="Store Securely" icon="save">
    Save it in an environment variable: `API_KEY=sk_test_your-sandbox-key-here`.
  </Step>
</Steps>

## Make Your First API Request

Test validation with a sample CURP (Clave Única de Registro de Población) in the sandbox. This endpoint checks against Mexican legal registries.

<Request tabs="JavaScript,cURL,Python" show-lines="true" default-tab="cURL">
  ```javascript
  const response = await fetch('https://api.apimarket.mx/sandbox/v1/validate/curp/HGUC900203HDFRNN07', {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer sk_test_your-sandbox-key-here',
      'Content-Type': 'application/json'
    }
  });
  const data = await response.json();
  console.log(data);
  ```
  ```bash
  curl -X GET "https://api.apimarket.mx/sandbox/v1/validate/curp/HGUC900203HDFRNN07" \
    -H "Authorization: Bearer sk_test_your-sandbox-key-here" \
    -H "Content-Type: application/json"
  ```
  ```python
  import requests

  url = "https://api.apimarket.mx/sandbox/v1/validate/curp/HGUC900203HDFRNN07"
  headers = {
      "Authorization": "Bearer sk_test_your-sandbox-key-here",
      "Content-Type": "application/json"
  }
  response = requests.get(url, headers=headers)
  print(response.json())
  ```
</Request>

<Response tabs="200" show-lines="false">
```json
{
  "success": true,
  "data": {
    "curp": "HGUC900203HDFRNN07",
    "valid": true,
    "name": "Carlos Hernández García",
    "birth_date": "2003-02-20",
    "registries": ["RFC", "INE", "SAT"]
  }
}
```
</Response>

<Callout kind="success">
Your first request succeeded! The sandbox uses mock data for safe testing.
</Callout>
