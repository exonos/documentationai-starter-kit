## Pasos para obtener tu API Key

Sigue estos pasos para acceder a las APIs gubernamentales:

1. Inicia sesión en [apimarket.mx](https://apimarket.mx)
2. Ve a tu perfil de usuario.
3. Selecciona "Mis APIs".
4. Haz clic en "Generar nueva clave".
5. Copia tu API Key y guárdala de forma segura.

## Uso en solicitudes

```bash
curl -H "Authorization: Bearer TU_API_KEY" https://api.apimarket.mx/v1/endpoint
```
